Los atributos en HTML

Los atributos proporcionan informacion adiccional sobre elementos HTML

Todos los elementos HTML pueden tener atributos

Los atributos proporcionan informacion adiccional sobre los elementos

Los atributos siempre se especifican en la etiqueta de inicio

Los atributos generalmente vienen en pares de nombre/valor como nombre ="valor"

Atributo href

la etiqueta a define un hipervinculo. El href atributo especifica la URL de la pagina a la que va el enlace

Atributo src source

La etiqueta img se utiliza para incrustar imagenes en una pagina web HTML, el src es el atributo que especifica la ruta de la imagen a mostrar

URL absoluta

Enlace a una imagen externa alojada en algun sitio web cualquiera Ejemplo: src="www.google/imagen.com" se recomienda no usar estas imagenes por derechos de autor o en un futuro se pueden cambiar por cualquier imagen

El atributo lang

Siempre hay que incluir lang para declarar el idioma de la pagina esto para ayudar a los motores de busqueda y navegadores

Los codigos del pais se escriben dentro de lang los dos primeros caracteres definen el idioma de la pagina y los dos ultimos caracteres del pais por ejemplo el que se puso en esta pagina HTML
